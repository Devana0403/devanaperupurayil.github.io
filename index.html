Motherese Hypothesis — Prosody-Aware Whisper

Extended OpenAI's Whisper encoder-decoder to jointly model lexical and suprasegmental prosodic features (pitch, energy, duration). Trained from scratch on LibriSpeech and CHILDES corpora and evaluated on BLiMP syntactic benchmarks, finding prosody yields up to +66% F1 on long-distance dependency tasks.

AI Safety · LLMs · Evaluation

Scalable Oversight via Adversarial Resume Screening

Modelled scalable AI oversight as a Guard–Houdini adversarial game in resume screening. Ran 25-game evaluations across 16 model pairings (GPT-4o, Gemini 2.0, Llama 70B). Found that Guard oversight performance stays flat as model capability grows — evidence that some domains resist scalable oversight.

Physics · ML · Bayesian Inference

MJD Data Release — Neutrinoless Decay Analysis

Applied MCMC Bayesian parameter estimation (emcee, 50 walkers × 10k steps) to the Majorana Demonstrator detector dataset to constrain the signal count of neutrinoless double beta decay. Achieved TPR = 0.801, AUC = 0.86, and a 90% confidence upper limit of θ_NLDBD < 7.97.

  • Python
  • emcee
  • SciPy
  • MCMC
  • Bayesian Stats
Manifold Learning · Spectral Methods

Noise-Adaptive Spectral Embedding for Diffusion Maps

Implemented and evaluated three data-driven truncation strategies for diffusion maps under additive noise across 120 seeded runs on 5 synthetic manifolds. Bandwidth stability outperformed the eigengap heuristic by +3.5 pp trustworthiness on the S-curve and showed zero seed variance on uniform-curvature manifolds.

  • Python
  • NumPy
  • Diffusion Maps
  • Spectral Methods
  • Manifold Learning
